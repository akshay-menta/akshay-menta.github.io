InfoMeme: AI-Driven Meme Digest for News Summarization

Developed an AI system that transforms trending news into engaging memes by blending NLP, emotion analysis, and multimodal generation

RoBERTa, T5, Gemini Pro, Streamlit, Hugging Face, LLM APIs

Team size: 2

Date: Jul 2025

Course: DS5983 Large Language Models

InfoMeme is an innovative AI-powered system that converts real-time news into digestible and shareable memes, making information consumption both entertaining and accessible. Key components:

  • News Article Processing: Fetches and processes news from multiple categories via NewsAPI or user-provided URLs.
  • Article Summarization: Uses T5-small to generate concise, informative summaries optimized for meme creation.
  • Fake News Detection: Employs fine-tuned BERT/RoBERTa models to filter out unreliable or low-credibility news sources.
  • Emotion Analysis: Applies a fine-tuned RoBERTa model for multi-label emotion classification to capture nuanced context.
  • Meme Generation Pipeline: Analyzes article summaries, generates context-aware captions with Gemini Pro, aligns sentiment with suitable templates, and creates final memes using ImgFlip API.
  • Streamlit Interface: Offers an intuitive web UI for real-time news-to-meme generation with meme previews and analytics.
This project demonstrates expertise in multimodal AI, fact-aware NLP, and creative human-AI interaction design.

EmoLingo ChatBot: Emotion-Aware & Style-Preserving Conversations

Built a personalized chatbot that predicts user emotion and writing tone to generate empathetic and stylistically aligned responses

T5, RoBERTa, ELECTRA, Streamlit, Hugging Face, LLM APIs

Team size: 3

Date: Apr 2025

Course: CS6120 Natural language Processing

EmoLingo is an intelligent chatbot system that integrates emotional and linguistic awareness for more meaningful human-AI conversations. Key components:

  • Emotion Classification: Fine-tuned T5 and RoBERTa models on the GoEmotions dataset to detect 28 nuanced emotions with macro-F1 optimization.
  • Writing Style Detection: Created a custom 12-class style dataset from Reddit & Google and fine-tuned RoBERTa for tone classification.
  • Response Generation: Used prompt engineering with LLaMA3-70B via Groq API to generate supportive replies mimicking user's emotional tone and writing style.
  • UI Integration: Deployed on Streamlit with real-time input, emotion/tone display, and toggle-based theme (light/dark).
Highlights:
  • Used ensemble modeling and LoRA for efficient fine-tuning
  • Achieved style-preserving response generation with high fluency and empathy
  • Modular NLP pipeline with emotion, tone, and proficiency integration
  • Explored applications for non-native English speakers in tech support environments
This project demonstrates advanced NLP, emotional intelligence, and human-aligned LLM interaction design.

Temperature Forecasting using DMD and Deep Learning Methods

Built a hybrid forecasting pipeline combining Dynamic Mode Decomposition (DMD) with deep learning models for time series temperature prediction

DMD, LSTM, GRU, RNN, Python, Matplotlib

Team size: 5

Date: Dec 2022

Course: Mathematics For Intelligent Systems

This project focused on enhancing temperature forecasting accuracy using both classical and deep learning approaches. Major components:

  • DMD Model: Implemented Dynamic Mode Decomposition to capture low-rank temporal structures in temperature data for short-term forecasting.
  • Deep Learning Models: Designed and trained LSTM, GRU, and RNN architectures to handle long-range dependencies in time series data.
  • Hybrid Comparison: Compared standalone DMD and deep models across metrics like RMSE, MAE, and visual trend matching.
  • Visualization: Created insightful plots to compare predicted vs. actual trends using Matplotlib for interpretability.
Highlights:
  • Demonstrated effectiveness of DMD for linear temporal patterns
  • Achieved robust long-range predictions using LSTM and GRU networks
  • Explored model stacking for enhanced forecast reliability
  • Reproducible pipeline with modular Python implementation
This project showcases the synergy between signal decomposition techniques and deep learning in climate-related forecasting tasks.
